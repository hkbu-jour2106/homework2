Q3. What is the difference between <b> and <strong> tags?

Ans: <b> is a style means "Bold", but <strong> is semantic, you can bind other style to strong.

Example Code:


<style>
strong { 
    color: red;
    font-weight: 100;
}
</style>

<br>
